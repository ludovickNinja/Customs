# CustomPlatform

Customer-facing quote request form prototype for custom order pieces.

## Current scope

- Priority quote request form
- Supports multiple design variations in one request (starts with 1 version)
- Enter key on text inputs moves focus to the next field
- Add/remove variation blocks
- Ongoing projects search + status filters (PO, customer name, quote #, reference #, status)

## Run locally

Open `index.html` in a browser.
